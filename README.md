Running the application:
  1. Clone the repository
  2. Run the Spring Boot Application in an IDE or use maven command
  3. Run the Frontend Client: "npm install" (Only needed first time) -> "npm run dev". Then client should be accessible at http://localhost:5173/
  4. Use the test file provided: EmployeesBackend/testFile.csv or upload your own.
  5. If you want to run the tests you can either do that from an IDE or use mvn/npm command.
